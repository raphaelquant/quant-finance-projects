# Quant Finance Projects — Python

## Overview
Python projects in quantitative finance: portfolio theory,
risk management, and financial modeling.
Built as part of a self-directed learning path toward
quantitative finance roles.

## Projects

### 1. Portfolio Simulator
- Computes return, volatility and Sharpe ratio
- Two-asset portfolio with correlation effect
- Based on Markowitz Modern Portfolio Theory

### 2. Real Market Data Analysis
- Downloads real stock data via yfinance
- Tickers: TotalEnergies (TTE.PA), L'Oréal (OR.PA), CAC40
- Annualized returns and volatility (×252 trading days)
- Normalized price chart (base 100)

### 3. VaR & Sharpe Analysis
- 95% historical Value at Risk on TTE.PA
- Risk-adjusted performance metrics
- Return distribution visualization (fat tails)

## Tools
Python | NumPy | Pandas | Matplotlib | yfinance

## Concepts Covered
- Modern Portfolio Theory (Markowitz)
- Capital Asset Pricing Model (CAPM)
- Value at Risk (VaR 95%)
- Sharpe Ratio & risk-adjusted returns
- Statistical distributions & fat tails
- Annualization (×252 trading days)
